The dotfiles I am currently using on Arch Linux

# Currently in use:
- xfce4
- i3-gaps
- pywal
- polybar
- neovim (with vim8 plugins)
- compton
- rofi (launcher/window swapper)
- kitty (terminal)
- libinput-gestures
- tlp (power management)
- stow + git (dotfile management)

- vifm (vim style file manager)
